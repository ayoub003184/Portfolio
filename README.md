# Ayoub Cherfaoui - Portfolio

AI Engineering student, full-stack developer, and problem solver. This is my personal portfolio website showcasing my work, experience, education, and side projects. Built with pure HTML/CSS/JS - no frameworks, no build steps.

## Features

- Fully responsive layout (mobile, tablet, desktop)
- Smooth scroll navigation with animated fade-ins
- Sidebar with social and email links
- Interactive job experience tabs
- Featured projects with live demo links
- Certifications and awards section
- Developer-centric dark color scheme

## Built With

- HTML5
- CSS3 (custom properties, flexbox, grid)
- Vanilla JavaScript
